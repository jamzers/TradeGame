# TradeGame
# 🎮 Infinite Pokémon Zoo

Welcome to the TradeGame (Based of Pokemon Go but super super simple) ! This is a custom, full-screen interactive desktop game built using Python. Explore the field, click on wild Pokémon to catch them, and check out your growing collection inside your backpack! 

Featuring some fan-favorites like Pikachu, Charizard, Gengar, Jigglypuff, Squirtle, and especially Arbok and Rattata! 🐍🐀

## 🚀 Features
* Dynamic Auto-Loader:The game engine automatically scans the folder and imports any `.png` files it finds. Want to add more Pokémon? Just drop their images into the folder!
* Infinite Spawning: Catching a Pokémon immediately triggers a new spawn somewhere random on the map.
* Live Inventory Tracking: Open your Backpack to view exactly how many of each species you've collected.
* Sleek CustomUI:Designed with a dark mode aesthetic using `customtkinter`.
